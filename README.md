# Football Scout

A c++ program that uses real-world data to scout undervalured football players.


## Run Locally

Clone the project
```bash
git clone https://github.com/Dennis-Gega/football_scout.git
```


Go to the project and setup build directory

```bash
  cd football_scout
  mkdir build && cd build
  cmake ..
  cmake --build .
```
Run the executable

```bash
  ./football_scout
```
